## 🚀 Quick Start (3 Steps)

### 1️⃣ Install Dependencies
```bash
cd ECO-Trade-app-main
npm install
```

### 2️⃣ Configure Environment
Create `.env` file:
```
PORT=3000
GEMINI_API_KEY=your_api_key
NODE_ENV=development
```

### 3️⃣ Start Server
```bash
npm run dev
```

**Server running at:** `http://localhost:3000`
